How to Add Another Player

Open this HTML file and scroll to the PLAYERS list near the bottom. Copy this example, paste it under Brayden, then update the info:

{
  name: "Player Name",
  grad: "2027",
  school: "High School Name (TX)",
  positions: "SS / RHP",
  heightWeight: "6'0 / 175",
  batsThrows: "R/R",
  sixty: "7.00",
  exitVelo: "88 mph",
  fieldLevel: "https://fieldlevel-link-here",
  ncsa: "https://ncsa-link-here",
  video: "https://youtube-or-video-link-here",
  notes: "Short recruiting note goes here."
}

Make sure every player block has a comma after it except the last player in the list.
